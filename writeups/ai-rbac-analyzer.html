AI RBAC Analyzer

Projekt för att analysera Azure RBAC med hjälp av AI, PowerShell och Ollama.

Syfte

Målet var att förstå RBAC och least privilege bättre i praktiken.

Teknik

  • Azure RBAC
  • PowerShell
  • Ollama
  • Llama3
  • JSON

Flöde

Azure RBAC → JSON export → PowerShell → Ollama → AI analysis

Lärdomar

  • Contributor-roller kan bli väldigt breda
  • Scope är kritiskt inom RBAC
  • AI kan hjälpa till att analysera access snabbare
