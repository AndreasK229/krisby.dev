Entra ID + Cloudflare Access + SCIM

Satte upp Entra ID som IdP mot Cloudflare Access för min domän för att förstå federation, provisioning och Zero Trust bättre i praktiken.

Arkitektur

Entra ID → OIDC → Cloudflare Access Entra ID → SCIM → Cloudflare

Detta används

  • Microsoft Entra ID
  • Cloudflare Access
  • OIDC federation
  • SCIM provisioning

Problem under setup

  • Redirect URI mismatch
  • Invalid client secret
  • API permissions
  • Admin consent

Lärdomar

  • SCIM förenklar provisioning och deprovisioning
  • Gruppbaserad access sparar mycket manuellt arbete
  • Det är mycket lättare att förstå authflöden när systemen kör live
